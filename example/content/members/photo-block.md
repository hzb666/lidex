The gallery page uses this lighter block type to prove that structured content is not only for text-heavy pages. The same page shell, asset serving, and template lookup mechanisms still apply.

That makes visual sections easier to keep inside the same project conventions. Assets live under the project root, while block templates decide how image cards should behave inside the shared theme.
